本政策说明我们在您使用 Askar Provisioning（本应用）时如何收集、使用、存储和保护与个人信息相关的数据。请您在使用前仔细阅读。若您不同意，请停止使用本应用。

1. 我们处理的信息与目的

  • 账户与安全：若您使用与服务器交互的功能，我们可能处理访问令牌及与认证刷新相关的数据，用于维持会话与安全（如适用）。
  • 设备配网与连接：为完成蓝牙（BLE）配网、Wi-Fi 扫描与连接，我们会处理设备标识、网络标识（如 Wi-Fi SSID）、您输入或授权的网络凭据等，仅用于实现配网与本地连接。
  • 本地网络与设备控制：本应用通过局域网与您配置的 Alpaca 兼容设备通信，以提供发现、状态与控制等功能；相关请求主要在本地网络内完成。
  • 应用更新：我们可能向运营方服务器发送应用标识（如包名）、平台与当前版本号，用于检查更新；具体字段以实际接口为准。
  • 本地偏好与注册信息：例如界面语言、用户名、您在本应用中登记的设备清单与分组等，用于改善使用体验与本地管理。

2. 系统权限

我们可能请求蓝牙、附近设备/Wi-Fi、网络与位置相关权限（用于在系统要求下完成 Wi-Fi 扫描或蓝牙通信，具体以系统提示为准）。您可在系统设置中管理权限；拒绝可能影响相关功能。

3. 存储与安全

敏感信息（例如访问令牌、按 SSID 存储的 Wi-Fi 密码）使用系统安全存储机制保存在您的设备上。其他偏好与设备注册信息可能使用本地常规存储。请妥善保管设备与系统锁屏。

4. 网络传输说明

本应用可能允许通过 HTTP 等非加密通道与指定服务器或设备通信（例如开发/内网联调环境或局域网设备）。在非可信网络环境下存在被窃听或篡改的风险；建议在信任的网络环境中使用，并在生产环境中由运营方改为 HTTPS 等更安全的方式。

配网过程中，本应用可能在本地临时开启 HTTP 服务接收设备上报的数据，该服务仅在配网流程需要时使用。

5. 第三方 SDK 与统计分析

以您实际下载的安装包及我们公布的清单为准。当前应用版本未集成典型的广告或通用三方统计分析 SDK（如 Firebase Analytics）。如后续增加，我们将在本政策与应用内显著位置更新说明并取得必要同意。

6. 跨境传输

原则上个人信息处理在您的终端及您所连接的网络环境中进行。若运营方服务器位于境外或发生跨境传输，我们将依法履行告知与同意义务（如适用），并可通过更新本政策说明接收方与安全措施。

7. 未成年人

若您为未成年人，请在监护人指导下阅读本政策并使用本应用。我们不会主动面向儿童进行营销 profiling；如无意中收集了儿童个人信息，请联系我们删除。

8. 您的权利

在适用法律允许的范围内，您可就个人信息行使查阅、复制、更正、删除、撤回同意、注销等权利。您可通过上文联系方式与我们联系；我们将在法定期限内答复。

部分信息仅存于您的设备本地，您也可通过卸载应用或清除应用数据的方式删除（请注意备份）。

9. 政策更新

我们可能适时修订本政策。重大变更将通过应用内提示或官网/Github Pages 公告等方式告知。若您继续使用，即表示接受修订后的政策。
